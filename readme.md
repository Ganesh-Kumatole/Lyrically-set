# Music Player

A simple web-based music player built with HTML, CSS, and JavaScript.

## Features

- Play, pause, skip forward, and skip backward between songs
- Volume control with mute/unmute icon
- Seek bar to scrub through the song
- Displays current time and total duration
- Responsive and modern UI
